## 1. Summary

The project delivers a lightweight, user-friendly alarm system that detects unauthorized access using motion sensing and triggers an audio siren. It targets student rooms and small spaces, emphasizing compact hardware, simple operation, and clear status display for quick arming/disarming and ease of use.

Delivered features:
- Motion detection with a dedicated sensor.
- Audible siren for intrusions.
- Arm/disarm state machine with single-button operation.
- Basic information display and compact physical design focus.
